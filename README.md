# 📝 ウェブアプリケーション（クライアント + 管理者ページ）


## 📌 プロジェクト紹介
---
このプロジェクトは、過去に制作したウェブサイトをベースにしたものです。
Reactを使用して制作したアプリで、投稿の作成・編集・削除ができ、管理者ページとクライアントページの2つで構成されています。


## 🛠 使用技術
---
- **React 18**
- **React Router DOM**
- **LocalStorage（データ管理）**
- **JavaScript (ES6+)**
- **CSS / Styled-Components**
- **Axios**
- **React-js-pagination**
- **Firebase（画像アップロード機能 / デモ版では無効）**


## ⭐ 主な機能
---
### 🔸 クライアント（ユーザー）
- 会社製品やサービス内容を紹介するページで構成。
- 最新のお知らせを確認できる「カスタマーセンター」ページを実装。

### 🔸 管理者ページ
- 管理者ログイン機能（簡易的な認証で、API を用いて ID / PW を確認する機能です）
- 投稿の作成・編集・削除（CRUD）が可能。
- クライアント側に表示されるお知らせの管理機能を提供。
- 画像アップロード欄（デモ版では無効）を設置。


## 📁 フォルダ構成
---
/project-root
 ├─ /client           # クライアント側

 └─ README.md         # プロジェクト全体説明


## 🚀 セットアップ
---
git clone https://github.com/bowoo96/react-web-portfolio.git
cd project-root

npm install
npm start


## 🚨 注意事項
---
- 管理者ページで画像アップロード機能は、ポートフォリオ公開の都合上 Firebase の設定を外しています。
- 管理者ログイン機能はデモ用のため、現在 API とは接続していません。
   そのため、どのような ID / パスワードを入力してもログインできます。
- 実案件では正常に動作するコードになっています。
